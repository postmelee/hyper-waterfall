# `orders/` 폴더 규칙

## 목적

오늘 무엇을 할 것인지와 현재 상태를 한 장으로 보여준다.

## 답하는 질문

- "지금 어떤 이슈를 진행 중인가?"
- "오늘 완료된 작업은 무엇인가?"

## 작성 시점

task 시작, 상태 변경, 최종 보고, merge 후 정리 시점.

## 허용 파일명

`yyyymmdd.md`만 허용한다.

## 사용 템플릿

`mydocs/_templates/orders.md`

## 반드시 포함할 내용

- 날짜 제목
- 마일스톤 섹션
- `Issue | 타스크 | 상태 | 비고` 표
- 완료 작업의 `완료: HH:mm`

## 두면 안 되는 내용

- 긴 계획서
- 검증 로그
- 기술 조사
- 트러블슈팅
- 날짜가 아닌 템플릿 파일

## 다음 세션 AI가 복원해야 할 맥락

오늘 이어서 해야 할 작업과 각 이슈의 현재 상태.
