# `orders/` Folder Rules

## Purpose

Show what will be done today and the current status on a single board.

## Questions Answered

- "Which Issue is currently in progress?"
- "What work was completed today?"

## When to Write

When a task starts, status changes, the final report is written, or post-merge cleanup runs.

## Allowed Filename

Only `yyyymmdd.md` is allowed.

## Template Used

`mydocs/_templates/orders.md`

## Required Content

- Date title
- Milestone sections
- `Issue | Task | Status | Notes` table
- `Done: HH:mm` for completed work

## Content Not Allowed

- Long task plans
- Verification logs
- Technical research
- Troubleshooting
- Template files that are not date files

## Context the Next Session AI Must Restore

Work that should continue today and the current state of each Issue.
