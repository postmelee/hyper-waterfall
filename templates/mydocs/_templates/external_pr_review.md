# 외부 PR 검토 템플릿

이 파일은 `mydocs/pr/pr_{번호}_review.md` 작성용 중앙 템플릿이다. 외부 기여자 PR 검토의 초기 판단, 영향 범위, 검증 계획, 권고를 기록하고 작업지시자에게 검토 방향 승인을 받기 위한 문서다.

## 사용 위치

- 실제 파일: `mydocs/pr/pr_{번호}_review.md`
- 작성 시점: 외부 기여자 PR 메타데이터와 diff를 확인한 직후
- 작성 언어: 한국어
