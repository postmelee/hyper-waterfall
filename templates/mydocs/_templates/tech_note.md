# 기술 조사 템플릿

이 파일은 `mydocs/tech/`에 기술 조사, 구조 분석, 스펙 검토, 대안 비교를 기록할 때 사용하는 중앙 템플릿이다. 기술 조사 문서는 특정 Stage의 완료 보고가 아니라, 이후 작업에서 반복 참조할 수 있는 근거와 결정을 남기는 용도다.

## 사용 위치

- 실제 파일: `mydocs/tech/{yyyymmdd}_{topic}.md` 또는 `mydocs/tech/task_{milestone}_{issue}_{topic}.md`
- 작성 시점: 구현 전 조사, 설계 대안 비교, 외부 문서/스펙 검토, 구조 분석 시점
- 작성 언어: 한국어
